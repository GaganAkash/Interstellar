# Bibliography for the Project

## External Libraries, Frameworks, and Resources

- **React**  
  A JavaScript library for building user interfaces.  
  [https://reactjs.org/](https://reactjs.org/)

- **React Router DOM**  
  Declarative routing for React applications.  
  [https://reactrouter.com/](https://reactrouter.com/)

- **Framer Motion**  
  A production-ready motion library for React.  
  [https://www.framer.com/motion/](https://www.framer.com/motion/)

- **React Icons**  
  Popular icons as React components.  
  [https://react-icons.github.io/react-icons/](https://react-icons.github.io/react-icons/)

- **Flask**  
  A lightweight WSGI web application framework for Python.  
  [https://flask.palletsprojects.com/](https://flask.palletsprojects.com/)

- **Flask-Login**  
  User session management for Flask.  
  [https://flask-login.readthedocs.io/](https://flask-login.readthedocs.io/)

- **Flask-CORS**  
  Cross-Origin Resource Sharing support for Flask.  
  [https://flask-cors.readthedocs.io/](https://flask-cors.readthedocs.io/)

- **Werkzeug**  
  A comprehensive WSGI web application library.  
  [https://werkzeug.palletsprojects.com/](https://werkzeug.palletsprojects.com/)

- **Tailwind CSS**  
  A utility-first CSS framework for rapid UI development.  
  [https://tailwindcss.com/](https://tailwindcss.com/)

- **PostCSS**  
  A tool for transforming CSS with JavaScript plugins.  
  [https://postcss.org/](https://postcss.org/)

- **IPFS (InterPlanetary File System)**  
  A peer-to-peer hypermedia protocol.  
  [https://ipfs.io/](https://ipfs.io/)

- **QR Code Generation Library**  
  Used in backend for generating QR codes for IPFS CIDs.
